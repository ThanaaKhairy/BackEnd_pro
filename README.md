# 🌍 Digital Nomad Visa Platform API

A backend RESTful API that helps users explore digital nomad visas, interact with the community, and manage advertisements. The system includes secure authentication, country management, social posts, and admin controls.

## 🚀 Features

### 🔐 Authentication

* User Registration & Login
* Email Verification (OTP)
* Forgot & Reset Password
* JWT Authentication
* Secure Password Hashing (bcrypt)

### 🌎 Countries Management

* View all countries and visa programs
* Search by country name or ID
* Add, update, and delete countries (Admin)

### 📝 Community Posts

* Create, update, and delete posts
* Like and unlike posts
* Add and manage comments

### 🛡️ Admin Panel

* Manage users
* Promote/Demote admins
* Delete accounts
* Monitor platform content

### 📢 Advertisement System

* View active advertisements
* Track ad clicks
* Create and manage ads (Admin)

## 🛠️ Tech Stack

* Node.js
* Express.js
* MongoDB
* Mongoose
* JWT
* bcrypt
* Zod
* Vercel

## 📖 About

A scalable backend API built for managing digital nomad visa information, community engagement, and advertisement services with a strong focus on security and clean architecture.
